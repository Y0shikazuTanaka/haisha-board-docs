# プライバシーポリシー

**最終更新日:** 2026年1月27日

## 1. はじめに
「ハイシャボード」（以下、本アプリ）は、[開発者名: 田中 嘉一]（以下、当方）が開発・提供するWindows用デスクトップアプリケーションです。
本アプリは、ユーザーのプライバシーを尊重し、個人情報の保護に最大限の配慮を行っています。本ポリシーでは、本アプリにおける利用者情報の取り扱いについて説明します。

## 2. 収集する情報とその利用方法

**本アプリは、ユーザーが入力した顧客情報、ドライバー情報、売上データなどの業務データを、当方のサーバーや第三者のサーバーに送信・収集することは一切ありません。**

すべてのデータは、お客様が使用されているPC内のローカルデータベース（SQLite）にのみ保存されます。

### 自動的に収集される情報
本アプリの配信プラットフォームであるMicrosoft Store、およびWindows OSの機能により、以下の匿名化された情報が収集される場合がありますが、これらは個人を特定するものではありません。

- アプリのクラッシュレポート（不具合の解析のため）
- アプリのインストール数、利用地域などの統計データ

## 3. データの保存と保護
本アプリで作成されたすべての業務データ（`haisha.db` ファイル）は、お客様のデバイス内に保存されます。
当方はこれらのデータにアクセスすることはできず、したがってデータのバックアップや復旧はお客様ご自身の管理下で行っていただく必要があります。

重要なデータについては、本アプリのバックアップ機能を利用し、定期的に外部ストレージ等へ保存することを強く推奨します。

## 4. インターネット接続について
本アプリは、以下の目的でのみインターネット接続を利用する場合があります。

- アプリケーションの更新確認
- Microsoft Storeのライセンス認証

これらの通信において、お客様が入力した業務データが送信されることはありません。

## 5. 免責事項
本アプリの使用によって生じた、いかなる損害（データの消失、業務の中断、利益の損失など）についても、当方は一切の責任を負わないものとします。データの管理はお客様自身の責任において行ってください。

## 6. プライバシーポリシーの変更
本アプリの機能追加や法令の変更に伴い、本ポリシーを改定することがあります。重要な変更がある場合は、本ページにて通知します。
